#### Tommy x2 no-flip

| **Pins** | **Simul-L**                          | **Simul-R**                       |
|----------|--------------------------------------|-----------------------------------|
| dl       | [DL] **(-d+c)**                      | [UR] **(-r+dr)+(-L+U)**           |
| R        | [UL] **(-r+d)**                      | [UR] U → L                        |
| DR       | [UL] UL → U,L                        | [DR] C → U,L                      |
| \        | [UL] **-[(-C+U)+D+(-l+ul)+(-r+dr)]** | [UR] **(-c+u)+d+(-L+UL)+(-R+DR)** |
| UL       | [UL] UL,U,L,C → D                    | [UR] DL → D                       |
| L        | [UL] UL,U,L,C,DL,D → R               | [UR] DR → R                       |
| ur       | [UL] UL,U,L,C,R,DL,D,DR → 12         | [UR] UR → 12                      |
